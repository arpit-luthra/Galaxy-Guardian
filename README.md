# 🚀 Galaxy-Guardian 

**Summary**
---
Galaxy Guardian is a fast-paced C# survival game where you navigate a spaceship to dodge asteroids that speed up over time. Built with .NET, it features dynamic difficulty and high-score tracking.

**Objective**
---
Survive as long as possible. Asteroids speed up every 10 seconds—collect shining stars for 2x points and red hearts for extra lives to beat your high score.

**Instructions & Controls**
---
**Move**: Arrow Keys (Supports horizontal screen wrapping)

**Power-ups**: Stars (2x Score), Red Hearts (Extra Life)

**Pause / Exit**: P to Pause, Esc to Exit

**Menu**: Backspace to return to the main menu

**How to Run**
---
* Download and extract the ZIP.
* Open GalaxyGuardian.sln in Visual Studio.
* Press Start to play.
